Note-Taker

Aplicação web para a criação e gerenciamento de notas mentais, o princípio fundamental é ter uma interface simples para a criação de notas mentais e posteriormente conseguir fazer pesquisas na base de notas criadas, relacionando assuntos e ajudando no raciocínio utilizando inteligência artificial, também será disponibilizada interface para gerenciamento das notas criadas.

Principais Funcionalidades:
  • Criação de notas mentais
  • Gerenciamento de notas criadas
  • Chat com inteligência artificial que é alimentada com as notas criadas
